# Neovim

## Prereqs

### Pacman requirements
```
sudo pacman -S go luarocks wget ruby fd ripgrep fzf xclip python3
```

### AUR requirements

```
paru -S nvm nvim-packer-git python-pynvim-git ruby-neovim
```

### NPM requirements
```
npm i -g eslint_d
```

## RUN CHECKHEALTH
run :checkhealth from inside nvim and fix other issues
